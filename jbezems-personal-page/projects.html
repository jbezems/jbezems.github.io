Full-Stack Web-Game: Nut-Flix and Chill

As the final project for EECS 493 User Interface Development, my team of four developed this full-stack web-based game to educate users about beloved Ann Arbor squirrels. Built in JavaScript and using jQuery, we fulfilled universal design principles and followed a complete design cycle, from storyboarding to prototyping to user testing to launching the final game. I executed game timing and levels, point tracking, and lives of the character. See my final presentation on this project below.

SillyQL: Relational Databases

This project emulated a basic relational database using an interface based on a subset of of a standard query language. A course project for EECS 281 Data Structures and Algorithms, I designed this relational database by making use of multiple interacting data structures, including hash tables and binary search trees. The user is able to use the subset of an SQL to perform functions such as create, insert, delete, and join on the database.
